Head

A tag head em HTML é onde você coloca os metadados sobre o documento HTML. Estes dados não são diretamente exibidos na página web, mas são usados para configurar, informar e melhorar a performance da página.

@charset "UTF-8"; --- é uma regra CSS que especifica a codificação de caracteres do arquivo de folha de estilo. Ele deve ser colocado na primeira linha do arquivo CSS e não pode ter nenhum caractere antes dele, incluindo comentários. Isso garante que o arquivo CSS seja interpretado corretamente pelo navegador.

link:css --- (CSS externo ) é um arquivo separado que contém regras CSS para estilizar um documento HTML. Para vincular uma folha de estilo CSS externa a um documento HTML, você pode usar o elemento link:css no cabeçalho do documento HTML. OBS.: utilidade para todas as páginas vinculadas.

style --- (CSS local) A tag é usada para definir estilos CSS, executada dentro do cabeçalho de uma página HTML. OBS.: utilidade para apenas uma página.

Propriedades

permitem controlar o estilo e o layout de elementos em uma página web.

!important ---usada para adicionar mais importância a uma propriedade/valor do que o normal, ela substituirá TODAS as regras de estilo anteriores para aquela propriedade específica naquele elemento!

Seletores

São formas de idêntificar os elementos HTML no css, facilitando a formatação.

*{} --- é um seletor universal em CSS que seleciona todos os elementos na página.

style="atributos"; --- forma de declarar atributos Css no formato inline, este formato de declaração é de escopo local, ou seja, aplica somente na tag onde a mesma foi declarada.

Tag name --- forma de identificar um elemento HTML pela Tag, esta forma é prioridade quando se trata de formatar um grupo de elementos que possuem a mesma tag HTML.

#ID{} --- Forma de identificara tag HTML pelo ID; esta forma é prioridade quando se trata de apenas um elemento HTML, pois é um indentificador exclusivo.

.class{} --- Forma de identificar elementos pela classe, esta forma é prioridade quando se trata de formatar um grupo de tags HTML que pertencem a mesma classe.

Há diferentes formas de declarar e especificar os seletores para formatação, os exemplos anteriores são as formas mais simples.
para mais informações e possibilidades há de pesquisar mais!

Propriedade background

formata o plano de fundo do elemento atribuído.

Shorthand background: color > image > repeat > position / size > attachment

background-color: altera a cor.

background-image: adiciona uma imagem.

background-repeat: formata a repetição do background.

background-position: posiciona o background.

background-size: define o tamanho da imagem de fundo do elemento.

background-clip: define até onde a cor ou a imagem de fundo de um elemento deve se estender.

background-attachment: define se a imagem de fundo de um elemento deve rolar ou permanecer fixa em relação à área de exibição.

Propriedade border

estilização da borda dos elementos HTML; É possivel personalizar cada lado da borda de forma única.

Shorthand border: color > style > width

border-style: formata o estilo da borda do elemento.

border-width: formata a espessura da borda.

border-color: define a cor da borda.

border-radius: personaliza o arredondamento das bordas.

border-radius: superior esquerdo, superior direito, baixo esquerda, baixo direita.

border-image

Shorthand border-image: URL > slice > width > outset > repeat

border-image: URL(); adiciona uma imagem na borda.

border-image-slice: Especifica como a imagem será dividida em partes para se ajustar às bordas.

border-image-outset: Aumenta a área da borda.

border-image-repeat: Define como a imagem será repetida (pode ser stretch, repeat, round).

propriedade font

Shorthand Font : style > variant > weight > size > family

@font-face {} --- é uma regra CSS que permite usar fontes personalizadas em seu site. Ela especifica uma fonte personalizada para exibir texto; a fonte pode ser carregada de um servidor remoto ou de uma fonte instalada localmente no computador do usuário.

src e format --- são descritores usados na regra @font-face. O descritor src é usado para especificar o local de onde a fonte deve ser carregada. Ele pode ser usado para especificar um URL para uma fonte remota ou para especificar o nome de uma fonte instalada localmente no computador do usuário. O descritor format é usado para especificar o formato da fonte que está sendo carregada. Ele é usado em conjunto com o descritor src para informar ao navegador qual o formato da fonte que está sendo carregada. Isso é útil porque diferentes navegadores suportam diferentes formatos de fonte.

TTF e OTF --- são formatos de fonte. TTF significa TrueType Font e OTF significa OpenType Font. Ambos são formatos de fonte escalonáveis, o que significa que eles podem ser redimensionados sem perder qualidade. TTF e OTF são suportados pela maioria dos navegadores modernos e podem ser usados com a regra @font-face para carregar fontes personalizadas em seu site. Para usar uma fonte TTF ou OTF com @font-face, você deve especificar o URL da fonte usando o descritor src e especificar o formato da fonte usando o descritor format. TTF = format('truetype') OTF= format('opentype') Existem outros formatos, porém esses dois são mais compatíveis. Além dos atributos font-family e src vistos anteriormente a diretiva @font-face contém ainda outros atributos opcionais: Font-stretch,; font-style; font-weight; unicode-range;

font-style --- A propriedade do CSS que define se uma fonte deve ser estilizada com uma face normal, itálica ou oblíqua de sua família de fontes.

font-variant --- A propriedade do CSS que especifica se um texto deve ser exibido em uma fonte small-caps ou não. Em uma fonte small-caps, todas as letras minúsculas são convertidas em letras maiúsculas. No entanto, as letras maiúsculas convertidas aparecem em um tamanho de fonte menor do que as letras maiúsculas originais no texto. A propriedade font-variant tem dois valores: normal e small-caps.

font-weight --- A propriedade do CSS que define o peso ou a intensidade da fonte (por exemplo, negrito). Os pesos disponíveis dependem da família de fontes que está atualmente definida. A propriedade font-weight tem vários valores, incluindo normal, bold, lighter, bolder e valores numéricos de 100 a 900.

font-size --- esta propriedade define o tamanho da fonte do texto de um elemento. Você pode definir seu valor como um valor absoluto (por exemplo, 12px, 14pt, 1cm) ou como um valor relativo (por exemplo, 1em, 1.2rem, 120%) Obs.: melhor medida é o 1em=16px.

font-family --- propriedade que define a fonte do texto de um elemento. Você pode definir seu valor como o nome de uma fonte específica ou uma lista de fontes em ordem de preferência, separadas por vírgulas. Se a primeira fonte da lista não estiver disponível no dispositivo do usuário, o navegador tentará usar a próxima fonte da lista e assim por diante, última fonte será a genérica. Ela é uma maneira de encurtar o código, especificando todas as propriedades individuais da fonte em uma única propriedade. A propriedade font é uma propriedade abreviada para:

line

Line-height --- espaçamento entre linhas de um documento HTML. A medida usa pode ser tamanho da letra 1em, 16px. (dica: centralizar conteúdo dentro de um container)

text-align ---propriedade que define o alinhamento horizontal do texto de um elemento. Você pode definir seu valor como left, right, center ou justify.

text-indent --- A propriedade do CSS que define a indentação da primeira linha em um bloco de texto. Pode ser definida por valores positivos ou negativos. Se o valor for positivo, a primeira linha será recuada para a direita.

Text-decoration

usada para adicionar uma linha de decoração ao texto usando as propriedades:.

line: propriedade é usada para adicionar uma linha de decoração ao texto. Ex.: overline , underline e line-through

color: propriedade é usada para definir a cor da linha de decoração

Style: propriedade define o estilo da decoração do texto (como sólido, ondulado, pontilhado, tracejado, duplo).
Pode ser medido em ex.: auto, px ou % : https://www.w3schools.com/cssref/pr_text_text-decoration.php

Thickness: propriedade especifica a espessura da linha de decoração.

Text-transform --- propriedade CSS text-transform específica como capitalizar um texto de um elemento. Pode ser usado para que o texto apareça com todas as letras maiúsculas ou todas minúsculas, ou com cada palavra em maiúscula.

padding

Shorthand padding: top, right, botton, left

padding: O atributo usado para definir o espaçamento interno de um elemento, ou seja, a área entre o conteúdo do elemento e sua borda.

Obs.: Quando é atribuido dois valores no atributo padding é respectivamente os lados horizontal, vertical

margin

Shorthand margin: top, right, botton, left

margin: propriedade que define o espaçamento externo dos elementos.

há possibilidade de configurar apenas uma lado da margem; ex.: margin-left: 10px

margin: auto; forma de centralizar os elementos filhos de um parent.

obs.: A centralização dos elementos só funcionará caso o width do elemento estiver definido.

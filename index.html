Head

A tag head em HTML é onde você coloca os metadados sobre o documento HTML. Estes dados não são diretamente exibidos na página web, mas são usados para configurar, informar e melhorar a performance da página.

@charset "UTF-8"; --- é uma regra CSS que especifica a codificação de caracteres do arquivo de folha de estilo. Ele deve ser colocado na primeira linha do arquivo CSS e não pode ter nenhum caractere antes dele, incluindo comentários. Isso garante que o arquivo CSS seja interpretado corretamente pelo navegador.

link:css --- (CSS externo ) é um arquivo separado que contém regras CSS para estilizar um documento HTML. Para vincular uma folha de estilo CSS externa a um documento HTML, você pode usar o elemento link:css no cabeçalho do documento HTML. OBS.: utilidade para todas as páginas vinculadas.

style --- (CSS local) A tag é usada para definir estilos CSS, executada dentro do cabeçalho de uma página HTML. OBS.: utilidade para apenas uma página.

Pseudo-classe

Pseudo-classes permitem aplicar estilos a elementos com base em estados ou características específicas que não são diretamente representados no HTML, como a posição de um elemento dentro de um contêiner.

:nth-child() --- seleciona elementos com base em sua ordem na hierarquia de seus irmãos. Por exemplo, se você tiver uma lista de itens, pode usar essa pseudo-classe para estilizar apenas determinados itens, como o primeiro, o segundo ou aqueles em posições ímpares ou pares.

permitem controlar o estilo e o layout de elementos em uma página web.

Seletores

São formas de idêntificar os elementos HTML no css, facilitando a formatação.

*{} --- é um seletor universal em CSS que seleciona todos os elementos na página.

style="atributos"; --- forma de declarar atributos Css no formato inline, este formato de declaração é de escopo local, ou seja, aplica somente na tag onde a mesma foi declarada.

Tag name --- forma de identificar um elemento HTML pela Tag, esta forma é prioridade quando se trata de formatar um grupo de elementos que possuem a mesma tag HTML.

#ID{} --- Forma de identificara tag HTML pelo ID; esta forma é prioridade quando se trata de apenas um elemento HTML, pois é um indentificador exclusivo.

.class{} --- Forma de identificar elementos pela classe, esta forma é prioridade quando se trata de formatar um grupo de tags HTML que pertencem a mesma classe.

Há diferentes formas de declarar e especificar os seletores para formatação, os exemplos anteriores são as formas mais simples.

propriedade font

Shorthand Font : style > variant > weight > size > family

@font-face {} --- é uma regra CSS que permite usar fontes personalizadas em seu site. Ela especifica uma fonte personalizada para exibir texto; a fonte pode ser carregada de um servidor remoto ou de uma fonte instalada localmente no computador do usuário.

src e format --- são descritores usados na regra @font-face. O descritor src é usado para especificar o local de onde a fonte deve ser carregada. Ele pode ser usado para especificar um URL para uma fonte remota ou para especificar o nome de uma fonte instalada localmente no computador do usuário. O descritor format é usado para especificar o formato da fonte que está sendo carregada. Ele é usado em conjunto com o descritor src para informar ao navegador qual o formato da fonte que está sendo carregada. Isso é útil porque diferentes navegadores suportam diferentes formatos de fonte.

TTF e OTF --- são formatos de fonte. TTF significa TrueType Font e OTF significa OpenType Font. Ambos são formatos de fonte escalonáveis, o que significa que eles podem ser redimensionados sem perder qualidade. TTF e OTF são suportados pela maioria dos navegadores modernos e podem ser usados com a regra @font-face para carregar fontes personalizadas em seu site. Para usar uma fonte TTF ou OTF com @font-face, você deve especificar o URL da fonte usando o descritor src e especificar o formato da fonte usando o descritor format. TTF = format('truetype') OTF= format('opentype') Existem outros formatos, porém esses dois são mais compatíveis. Além dos atributos font-family e src vistos anteriormente a diretiva @font-face contém ainda outros atributos opcionais: Font-stretch,; font-style; font-weight; unicode-range;

font-style --- A propriedade do CSS que define se uma fonte deve ser estilizada com uma face normal, itálica ou oblíqua de sua família de fontes.

font-variant --- A propriedade do CSS que especifica se um texto deve ser exibido em uma fonte small-caps ou não.

small caps (ou versalete), são versões em caixa alta das letras minúsculas, mas com altura menor que as letras maiúsculas regulares. Elas são frequentemente usadas para criar um efeito elegante e sofisticado, como em subtítulos, cabeçalhos ou acrônimos.

font-weight --- A propriedade do CSS que define o peso ou a intensidade da fonte.

Os pesos disponíveis dependem da família de fontes que está atualmente definida. A propriedade font-weight tem vários valores, incluindo normal, bold, lighter, bolder e valores numéricos de 100 a 900.

font-size --- esta propriedade define o tamanho da fonte do texto de um elemento.

Você pode definir seu valor como um valor absoluto (por exemplo, 12px, 14pt, 1cm) ou como um valor relativo (por exemplo, 1em, 1.2rem, 120%) Obs.: melhor medida é o 1em=16px.

font-family --- propriedade que define a fonte do texto de um elemento.

Você pode definir seu valor como o nome de uma fonte específica ou uma lista de fontes em ordem de preferência, separadas por vírgulas. Se a primeira fonte da lista não estiver disponível no dispositivo do usuário, o navegador tentará usar a próxima fonte da lista e assim por diante, última fonte será a genérica.

Preenchimento gradiente

é uma transição de cores que pode ser usada para criar efeitos de degradê em sites. É possível controlar a direção, a forma, as cores e como elas transitam de uma para outra.

linear-gradient: degradê de cores de forma linear.

exemplo:background: *repeating-linear-gradient(to top right, blue, red, green);

entre parênteses informe a direção e a quantidade de cores que o degradê linear irá possuir; Caso deseja repetir informe antes a propriedade repeating.

há possibilidade de definir em qual a posição onde cada cor irá iniciar informando a "%" da area definida.

gradiente-radial: degradê de cores em forma circular.

exemplo:background: radial-gradient(circle , blue, red, green);

O gradiente-radial funciona quase da mesma forma do gradiente-linear, apenas omite a direção das cores.

caso a area não seja proporcional, há possibilidade de informar um atributo exclusivo "circle".

gradiente no texto

a possibilidade de colocar gradiente no texto, porém não é de forma regulamentada pelo w3c, utilizando métodos alternativos.

                                
                                    h1{
                                        background: linear-gradient(to right, blue, red);
                                        -webkit-text-fill-color: transparent;
                                        -webkit-background-clip: text;
                                    }

primeiramente, aplica o gradiente no elemento html contida por o texto desejado.

-webkit-fill-color: transparent; serve para transformar o texto do elemento em transparente.

-webkit-background-clip: text; serve para cortar o background do elmento desejado.

como o texto esta em transparente, ele recebe a cor de fundo aplicado, e com a 3ª propriedade do exemplo recorta o fundo, permitindo apenas o texto transparent.

Shadow

text-shadow: é uma propriedade que aplica sombras nos textos.

Shorthand text-shadow: 3px 3px 6px rgba(0, 0, 0, 0.7);

os dois primeiro valores informam o deslocamento da sombra nos eixos X, Y.

o terceiro valor informa o raio do desfoque, que suaviza a sombra.

o quarto valor define a cor da sombra.

caso queira mais de uma sombra por texto, acrescente uma virgula e informe outros valores.

box-shadow: propriedade que define a sombra de caixa dos elementos.

A propriedade box-shadow se apropria das mesmas regras e atributos da propriedade text-shadow, porém, há um atributo que pode ser complementado o inset.

inset é o atributo que define as sombras internas da caixa dos elementos.

Transform

Propriedade CSS é usada para aplicar transformações a um elemento, como rotações, escalas, deslocamentos e inclinações. É uma ferramenta poderosa que permite modificar a aparência de um elemento de maneira visualmente interessante e interativa.

Shorthand: transform:translate(100px, 50px) rotate(45deg) scale(2);

transform: translate(X, Y) --- valor da propriedade transform que desoloca elemento pelo eixo X, e pelo eixo Y; a partir da posicão inicial do elemeto.

valores negativos invertem o deslocamento do eixo.

transform: rotate(deg) --- valor da propriedade transform que rotaciona o elemento utilizando a unidade de mediad deg.

transform: scale(2) --- valor da propriedade transform que escala o tamanho proporcional ao tamanho original do elemento, neste caso a unidade de medida da escala é 2.

transform: skewX(15deg) or skewY(15deg) --- valor da propriedade transform que inclina o elemento a partir do eixo (x ou y), utilizando unidade de medida deg.

transform-origin: 0% 0%; --- propriedade transform-origin define o ponto de referência do elemento para orientação da propriedade transform.

o valor da propriedade transform-origin é orientada pela unidade de medida %, respectivamente eixo X, eixo Y.

ponto de referência do elemento

0% 0% = ESQUERDA SUPERIOR

100% 0% = DIREITA SUPERIOR

100% 100%= DIREITA INFERIOR

0% 100%= ESQUERDA INFERIOR

50% 50%= MEIO DO ELEMENTO

Position

Propriedade CSS que controla como um elemento é posicionado dentro de uma página web.

static --- O elemento segue o fluxo normal da página, sem ser afetado por outras propriedades de posicionamento. É a posição padrão.

fiexed --- utilizada para fixar um elemento na tela, impedindo que ele se mova quando o usuário rola a página. Ele se posiciona em relação à viewport (área visível da janela do navegador) e permanece na mesma posição, mesmo com a rolagem.

Caso um elemento sobreponha outro, a propriedade z-index resolve o conflito de qual elemento será visível.

sticky --- O elemento é posicionado relativamente até que o ponto de rolagem especificado seja alcançado. Após esse ponto, ele se comporta como um elemento fixed.

relative --- O elemento é posicionado em relação à sua posição ao container parent.

absolute --- O elemento é removido do fluxo normal e posicionado em relação ao seu primeiro ancestral com um valor de position diferente de static.

Além de position, os atributos top, bottom, left e right são usadas para ajustar a posição de um elemento quando ele não está em static.

Display

propriedade é usada para especificar como um elemento é mostrado em uma página da web. Cada elemento HTML tem um valor de exibição padrão, dependendo do tipo de elemento.

display: none Remove o elemento da página visualmente e do fluxo do documento.

display: inline os elementos ocupam a mesma linha, e não adiciona a formatação.

display: block os elementos não ocupam a mesma linha.

display: inline-block os elementos ocupam a mesma linha e mantém a formatação.

display: table transforma o elemento em um "container de tabela" que atua como um table no HTML. Isso significa que ele define o contexto para comportar outros elementos que podem ser configurados como linhas (display: table-row) e células (display: table-cell), simulando o comportamento de uma tabela sem precisar usar diretamente uma estrutura de tabela no HTML.

Principais características do displau: table

Cria um modelo de tabela para layout, permitindo que você organize elementos de forma semelhante a uma tabela.

Funciona bem para layouts responsivos.

Usa elementos filhos com propriedades como table-row e table-cell para estruturação.

display: table-cell usada para estilizar elementos HTML como células de uma tabela, mesmo quando eles não estão dentro de um elemento table. Ao aplicar essa propriedade, o elemento passa a se comportar como uma célula de tabela (td), permitindo que você aproveite características como alinhamento, tamanho automático com base no conteúdo, e alinhamento de bordas com outros elementos configurados como tabelas.

atributos da propriedade table-cell

Comportamento similar a células de tabela: O elemento se ajusta automaticamente ao tamanho do conteúdo, como uma célula em uma tabela HTML tradicional.

Alinhamento e estrutura: Você pode usar propriedades como vertical-align para alinhar o conteúdo na célula.

Contexto de tabela: Funciona melhor quando usado dentro de elementos com display: table ou display: table-row, simulando uma tabela.

display: flex é uma propriedade super versátil do CSS que ajuda a criar layouts flexíveis e responsivos. Ao usar essa propriedade em um contêiner, os elementos filhos (itens flexíveis) podem ser organizados de maneira eficiente.

Principais características do display-flex

Direção:

Use flex-direction para definir o fluxo dos itens (row, column, row-reverse, column-reverse)

alinhamento:

justify-content alinha os itens horizontalmente (ex.: center, space-between, space-around).

align-items alinha os itens verticalmente (ex.: center, stretch, baseline).

Quebra de linha:

flex-wrap controla se os itens devem quebrar para a próxima linha (ex.: nowrap, wrap).

Proporção:

flex-grow controla o crescimento dos itens (ex.: 0, 1).

flex-shrink controla a redução dos itens (ex.: 0, 1).

flex-basis define o tamanho inicial dos itens (ex.: auto, 100px).
